# 👋 Comece aqui — em 3 passos

Você não precisa entender de código. **O Claude vai conduzir você no resto.**

---

## Passo 1 — Baixa o projeto

1. Sobe no topo desta página no GitHub
2. Clica no botão verde **`Code`** (canto direito)
3. Clica em **`Download ZIP`**
4. Vai pra pasta de Downloads e **descompacta o ZIP** (clique duplo no Mac, botão direito → "Extrair tudo" no Windows)
5. Move a pasta descompactada pra um lugar fácil de achar (ex: Documentos)

---

## Passo 2 — Abre no Claude Code Desktop

Se você ainda **não tem** o Claude Code Desktop:
- Baixa em https://claude.com/download
- Instala normalmente (Mac ou Windows)

### 🪟 Se você usa Windows: instala também o Git for Windows

O Claude precisa de um terminal compatível pra rodar comandos. No Mac já vem de fábrica, no Windows você precisa instalar:

1. Baixa em https://git-scm.com/download/win
2. Roda o instalador clicando **"Next" em tudo** (as opções padrão estão certas)
3. Isso instala o **Git Bash**, que é o que o Claude Code Desktop vai usar

(Usuário Mac/Linux pode pular este passo.)

### Abrindo o projeto

Com o Claude Code Desktop aberto:
1. **Arrasta a pasta do projeto** pra cima da janela do Claude Code
   *(ou: menu **File → Open Folder** → seleciona a pasta)*
2. O Claude Code vai abrir um chat dentro da pasta

---

## Passo 3 — Manda a primeira mensagem

Na caixa de chat, **digita exatamente isto**:

```
oi, vamos começar
```

Pronto. **O Claude vai te guiar em todo o resto**: pegar sua chave Gemini, instalar o que precisa, configurar sua marca, e criar seu primeiro anúncio. Tudo conversando.

---

## ❓ Se algo der errado

Não tenta consertar sozinho. **Manda a mensagem de erro no chat** e o Claude resolve.

Tipo:

> "Apareceu este erro: [cola o erro aqui]"

ou simplesmente:

> "Não tô conseguindo, tá dando erro"

O Claude foi treinado pra resolver os problemas comuns sem você ter que entrar em terminal nenhum.

---

## ⏱️ Quanto tempo leva?

- **Primeira vez** (instalação + setup): ~10 minutos (a maior parte é download do navegador headless do Playwright, ~200MB)
- **Próximas vezes**: 0 segundos. Abre a pasta, manda mensagem, cria criativo.

## 🤔 Tem que pagar alguma coisa?

- **Claude Code Desktop**: vem com sua assinatura Claude
- **Chave Gemini**: **grátis até 1500 usos por dia** (mais que suficiente pra criar uns 50 carrosséis por dia)
- Se passar disso, espera 24h ou aumenta o limite no Google AI Studio
